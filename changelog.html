Changelog

Version Alp-Y25R01

December 6, 2025

We’re returning to Alpha! A brand-new engine, a fresh versioning system, and a rebuilt foundation for the future.

Back to Alpha

Due to major internal changes, we’re officially returning to the Alpha stage. The engine migration required us to rewrite the entire project from scratch to provide a stable foundation.

Why a new engine? (Godot)

Our previous engine was not well-suited for the features we needed. Switching to Godot makes the application run smoother, faster, and more reliably. This is the final engine switch!

New Versioning System

Format: Alp-Y25R01

  • Alp-: Alpha release.
  • Y25: Release year (2025).
  • R01: Revision number.
  • a/b/c: Bugfix iterations.

Current Features

  • Draw (LMB), Erase (RMB).
  • Select 1 of 10 blocks (Keys 1-9, 0).
  • Pick block under cursor (Scroll Wheel).
  • Save/Load .mcpaint projects.
  • Clear canvas & Export to PNG.

Implemented (Coming to UI Soon)

  • Custom canvas size (up to 999x999).

Version 1.0

July 29, 2025

Full Release! New Engine, New Features!

rewritten from Unity to Python (PyGame) for faster development. Introducing "Ice Simple UI".

New Features

  • None Tool: Safe navigation mode.
  • Dedicated Eraser: Now a separate tool.
  • Fill Tool: Place or Remove blocks in an area.
  • Improved Inventory: Categories & Search bar.
  • Grid & Rulers: Helper overlays (not visible in export).
  • Layer System: Create complex scenes.
  • Custom Canvas Size: 2x2 to 48x27 blocks (in this version).
  • Special Objects: Sun & Moon (dependent on Time settings).
  • Weather Effects: Rain and Snow.
