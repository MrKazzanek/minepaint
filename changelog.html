Version 1.0.0 – July 29, 2025

Full Release! New Engine, New Features, Improvements, and more!

We're thrilled to announce our return and welcome you to the full release of MinePaint! This version is a landmark achievement, as we've switched the program's engine to allow for faster development and the implementation of highly-requested features.

The New Engine:

The program has been completely rewritten from the Unity engine to Python (using the PyGame framework) to enable the implementation of new and improved features!

Advantages of the New Engine:

  • Full Development Potential: The new foundation allows us to grow the program without the previous limitations.
  • Feature Possibilities: Nearly any feature is now possible to implement, such as the long-awaited layer system.
  • Simpler, Cleaner Code: A more organized codebase means faster development and fewer bugs.

Unavoidable Downsides:

  • A slightly simpler GUI is necessary for performance.
  • Higher system load (specifically, the application may take longer to start). We have optimized this as much as possible, and we apologize for any inconvenience.

The New "Ice Simple UI":

The engine change required a complete UI overhaul. Introducing "Ice Simple UI" (ISU) – a clean interface with soft edges and subtle highlights, designed for simplicity and performance. We chose the name because its primary light-blue color reminds us of Minecraft's ice block. We have no immediate plans to change the UI, but we don't rule it out for the future.

New Features:

  • None Tool: A safe, default-selected tool that performs no action. It allows you to safely navigate your canvas and point things out without accidentally placing or removing blocks.
  • Dedicated Eraser Tool: Previously integrated into the Pen tool (via right-click), the Eraser is now its own tool for convenience. You can adjust the eraser size from 1 to 10.
  • Fill Tool: A tool for rapid drawing. Hold the left mouse button to select an area, and right-click to confirm the action. It has two modes:
    • Place: Fills the selected area with the chosen block.
    • Remove: Clears all blocks from the selected area on the active layer.
  • Improved Inventory: The block inventory is now organized into categories for easier searching. It also includes a search bar (use Ctrl+A+Backspace to quickly clear your search).
  • Grid Overlay: A toggle button in the top-right corner now enables/disables a helper grid on the canvas. The grid is for reference only and will not appear in saved PNG images.
  • Canvas Rulers: Next to the grid button, a new toggle enables rulers that display the block count along the X and Y axes. These are also not visible in saved images.
  • Layer System: The highly-anticipated layer system is here! Create complex scenes that were previously impossible. The layer manager is located to the right of the hotbar.
    • Use the `+` and `-` buttons to add or remove layers.
    • Use the `^` and `v` buttons to reorder the selected layer.
    • The active layer is highlighted, and the project saves your currently selected layer.
  • Custom Canvas Size: Specify a canvas size on startup or via the 'Resize' button in Options. The size can range from 2x2 to a maximum of 48x27 blocks.
  • Special Objects (Sun & Moon): Found in the 'Special' panel, you can now add a Sun and Moon as background objects. Their availability depends on the 'Time' panel settings:
    • Day: Only the Sun is available.
    • Night: Only the Moon is available. Right-click while placing to cycle through its phases.
    • Own: Both the Sun and Moon can be placed.
  • Custom Background (Time: Own): When you select the 'Own' option in the 'Time' panel, you can define a custom gradient background by providing two HEX color codes.
  • Weather Effects: Add atmospheric effects to your scene! Weather is a background effect with three options: Clear (default), Rain, and Snow.

New Blocks:

A significant number of new blocks, many with unique blockstates, have been added to the program, which were not available in the old MinePaint.

Improved Features:

  • Visible Brush & Eraser Size: When using a brush or eraser size larger than 1, a preview of the affected area is now visible on the canvas, showing exactly what you will place or erase.
  • Clearer Inventory Tooltips: To see a block's name in the inventory, simply hover over it. The name will now appear clearly at the bottom of the UI.

Version 0.4.0 – March 19, 2025

New blocks, new features, UI improvements, a loading screen, and more!

New Blocks & Variants:

  • Added a new "Moss Blocks" category, introducing a moss block and a pale moss block.
  • Added a new category for "Shrieker Blocks" with multiple light-level states.
  • Added new Log variants: Dark Oak and Pale Oak.
  • Added new Leaves variants: Dark Oak and Pale Oak.
  • The Grass Block now has a new blockstate: Snowy Grass.
  • All leaf blocks can now be waterlogged using the block debugging tool.

New Features:

  • Quick Block Selection: You can now rapidly select blocks within a category by hovering over them and pressing keys 1-9. A control hint will appear to guide you.
  • Quick Blockstate Change: When using the block debugging tool, you can now middle-click a block on the canvas to cycle through its special states (currently available for Shrieker blocks).
  • New Confirmation Screen: A warning screen now appears for confirmation before clearing the entire canvas.
  • Loading Screen [EXPERIMENTAL]: A new studio splash screen now appears when the project starts.

Improvements & Texture Changes:

  • The "Control Hints" feature has graduated from experimental status and is now a core feature.
  • The texture of the inventory close button has been updated to be more consistent with the UI design.

Bug fixes:

  • Fixed a visual bug where the highlighting on the "New/Save/Load" buttons was not visible. (Bug #007)
  • Switching from the brush to the debugging tool no longer unintentionally changes a block. (Bug #008)
  • The inventory can now be correctly opened with the "E" key immediately after saving or loading a project. (Bug #009)

Technical changes:

  • Implemented a new backend system for handling complex key combinations.
  • Optimized the script for creating a new canvas, removing unnecessary processes.

Version 0.3.1 – March 17, 2025

Patch, technical, and optimization update!

Technical changes:

  • Added extensive internal code comments to improve future development speed.
  • Refactored the internal logic of the quick access inventory bar for better stability and performance.
  • The air block is no longer treated as a standard item in the quick access bar.

Optimization:

  • Removed unnecessary code to improve overall performance.
  • Deleted unused and redundant textures to reduce the application's file size.

Bug fixes:

  • Increased the size of control hints for better readability in all cases. (Bug #005)
  • Fixed a visual bug where the entire UI would disappear while holding down the clear canvas shortcut ('C' + 'N'). (Bug #006)

Version 0.3.0 – March 16, 2025

An update adding ores, stones, and many new features, including experimental control hints!

New Blocks:

  • Added stone variants: Andesite, Diorite, Calcite, Tuff, Granite.
  • Added stone ores: Copper, Coal, Iron, Lapis, Redstone, Gold, Emerald, Diamonds, and placed them in the "Ores" category.
  • Added Deepslate block with 3 states.
  • Added deepslate ores: Copper, Coal, Iron, Lapis, Redstone, Gold, Emerald, Diamonds, and placed them in the "Deepslate Ores" category.
  • Added Bedrock block.
  • Added Cobweb block.
  • Added Lava block with two states.

New Features:

  • Block List - A new UI element in the inventory that shows all blocks belonging to a hovered category (e.g., all Dirts, Ores, Leaves).
  • Easier Block Switching - You can now switch blocks within a category by clicking the side previews, not just the central one.
  • Day/Night Toggle - You can now switch between day and night using the "T" key.

Experimental Feature:

  • Control Hints - Helpful tooltips now appear when hovering over the brush tool, size options, or the block preview area.

Texture Changes:

  • The block preview area has been visually updated to better match the overall app design.
  • The day/night toggle button now features improved icons for both states.

Bug fixes:

  • Fixed a bug where the changelog button could be pressed while other windows (like inventory) were open.
  • Fixed a bug where the brush size would reset after a specific sequence of actions involving the debug stick and inventory. (Bug #004)

Technical changes:

  • The brush size setting is now saved more reliably across all actions.
  • Internal data structures were updated to support the new Block List feature.
