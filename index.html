What is MinePaint?

MinePaint is a powerful tool designed for drawing 2D images, panoramas, schematics, floor plans, and more using Minecraft-style blocks. Whether you are planning a massive build or creating pixel art, MinePaint gives you the canvas you need.

With an intuitive interface, a robust engine, and built-in layers, it’s the perfect companion for every digital architect.

Tools & Features

Everything you need to craft your masterpiece.

Pen

Standard drawing tool for placing blocks precisely.

Fill

Instantly fill or clear a selected area with blocks.

Picker

Quickly select a block from the canvas to use.

Eraser

Remove blocks easily. Includes size adjustment.

None

Safely navigate without editing the canvas.

Layer System

Create complex scenes with multiple layers. Manage visibility and order with ease.

Smart Hotbar

Quick access to your last 9 used blocks.

Custom Size

Canvas resizing up to 300x300 blocks.
