Saving and Loading

The program allows saving and loading files in the .mcpaint format, as well as exporting canvases to .png files.

Button: Save as

Allows you to save the canvas as a new .mcpaint file. After clicking, a window will open where you can choose the save location and enter the file name.

Button: Save

Allows you to save the canvas in a .mcpaint file. If the canvas was previously saved as a .mcpaint file, this button overwrites the changes in that file. If the canvas has not been saved before, this button works like Save as – it opens a window to choose the save location and file name.

Button: Open

Allows you to load a previously saved canvas for editing. Clicking it opens the file selection window. The loaded file must be in .mcpaint format.

Button: Export png

Allows you to export the canvas as an image in .png format. Clicking it opens a window where you can specify the save location and file name.
