Efficiency Features

The program includes several features that make using it easier.

Button: Minimize

Allows you to minimize the program window.

Canvas Drawing Enhancements


Canvas Grid

The # button toggles the canvas grid on or off. The grid helps with drawing and is not visible in exported .png files. The grid’s on/off state is saved. By default, the grid is off.


X and Y Axis Rulers

The R button toggles the canvas rulers on or off. This feature has several modes that can be changed by clicking the R button:

  1. Off – rulers are hidden.
  2. Spaced – rulers are visible, marked every 5 blocks.
  3. Precise – rulers are visible, marked for every single block.

By default, the rulers are off. The visibility state and marking type are saved.
