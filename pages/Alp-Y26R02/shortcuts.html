Keyboard Shortcuts

Below are useful keyboard shortcuts that improve your workflow in the program:

Tool Selection Shortcuts:

Shortcut Action
n Selects the None tool
p Selects the Pen tool
f Selects the Fill tool
c Selects the Picker tool
x Selects the Eraser tool
e Opens or closes the block inventory; also closes the blockstates inventory
ESC Closes the open inventory or blockstates panel

Canvas Management Shortcuts:

Shortcut Action
Ctrl + n Clears the canvas
Ctrl + Shift + r Opens the canvas resize window
Ctrl + Shift + s Saves the canvas as a new .mcpaint file.
Ctrl + s Saves the canvas as a new file or overwrites the existing .mcpaint file.
Ctrl + p Opens the canvas export window to .png
Ctrl + l Opens the load canvas window

Tool Management Shortcuts:

Shortcut Action
Ctrl + Shift + = Increases the brush size by 1.
Ctrl + = Resets the brush size to the default value (1).
Ctrl + - Decreases the brush size by 1.

Other Keyboard Shortcuts:

Shortcut Action
F2 Takes a screenshot of the entire program and saves it in the folder C:\Users\[Username]\Pictures\Screenshots\MinePaint_Screenshots
Ctrl + Shift + p Opens the folder with program screenshots: C:\Users\[Username]\Pictures\Screenshots\MinePaint_Screenshots.
