Pack file

Plik pack.json jest taką jakby instrukcją dla programu, która mówi mu że ten folder jest AssetsPackiem. Ten plik jest wymagany by AssetsPack mógł być wykrywany i mógł działać.

Wszystkie klucze pliku pack.json

Klucz Do czego służy?
name Definiuje nazwę AssetsPacka. Ten klucz jest wymagany.
description Opcjonalny klucz, umożliwiający dodanie opisu swemu AssetsPackowi
version Opcjonalny klucz, umożliwiający dodanie wersji swojego AssetsPacka

Zawartość pliku pack.json

Plik pack.json jest formatem zapisu danych oparty na strukturze JSON. Oto przykładowy pack.json:

{
    "name":"VanillaAssetsPack",
    "description":"Default assets from MinePaint as AssetsPack | By QozaWorks",
    "version":"Alp-Y26R02"
}
