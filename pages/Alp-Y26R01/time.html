Time System

The time system is responsible for setting the canvas background color. The background is created from two colors that form a vertical gradient (from top to bottom).

How to use the time system?

The program includes a Time panel, where you can choose one of three options:

  • Day – sets the daytime Minecraft sky colors. Colors used: Top → #7cabff, Bottom → #8cb5ff
  • Night – sets the nighttime Minecraft sky colors. Colors used: Top → #0a0d17, Bottom → #0b0b14
  • Own – allows you to set your own background colors.

How to set your own canvas background color?

Click the Own button in the Time panel. A window will open where you can enter two colors in HEX format: Top and Bottom. These colors will be used to create a background gradient.
