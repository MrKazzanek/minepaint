Objects System

The objects system allows you to add one of two objects: Sun or Moon. These objects are always displayed beneath all layers, regardless of their order. You can place them anywhere on the canvas. A maximum of one object – either the sun or the moon – can exist on a single canvas.

The objects are located in the Objects panel, where you can choose one of the following options:

  • Clear – removes the object from the canvas
  • Sun – allows you to place the sun
  • Moon – allows you to place the moon

How to place an object?

To place an object, select one of the available objects from the Objects panel. This will activate object placement mode. You can then place it where the mouse pointer is on the canvas.

  • Scroll Up – increases the object’s scale
  • Scroll Down – decreases the object’s scale
  • Right Mouse Button – for the sun: no special action; for the moon: changes the moon phase
  • Left Mouse Button – places the object on the canvas

Moon Phases

The moon has 8 phases, which can be changed in object placement mode by clicking Right Mouse Button.
