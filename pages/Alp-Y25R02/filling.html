Filling

The Fill tool, available in the toolbar, is used for filling areas in the program. It allows you to quickly fill a selected area with a chosen block or completely clear it.

How to use the tool?

Left Mouse Button – fills the selected area with the currently selected block.

Right Mouse Button – removes the contents of the selected area (clears it).

Filling Process

Click with the left or right mouse button to set the first point on the canvas. Then click with the same button to set the second point. The area between the selected points will be filled or cleared accordingly.
The tool works by selecting two opposite points that define the operation area.
