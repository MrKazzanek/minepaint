Layers System

The program includes a canvas layers system that allows you to manage drawing elements in the Layers panel.

How does the layers system work?

By default, the program starts with a single layer. New layers can be created using the + button in the Layers panel. A newly created layer appears above the currently active layer. The active layer is the one the user is currently working on. In the Layers panel, the active layer is highlighted in a light blue color, and a small red square next to its name also indicates that it is active. The selected (active) layer can be removed using the - button in the Layers panel. If there is only one layer left, it cannot be removed. The order of layers in the list corresponds to their order on the canvas: the higher a layer is in the panel, the higher it appears on the canvas; the lower it is, the lower it appears. The active layer can be moved up or down using the arrow buttons in the Layers panel. Additionally, each layer can be temporarily hidden, allowing you to work on other parts of the canvas without the layer being visible.

You can draw/erase, etc., only on the active layer.
