Welcome to MinePaint

MinePaint is a powerful external tool designed for drawing 2D images, panoramas, and schematics using blocks. Whether you are planning a massive build or creating pixel art, MinePaint gives you the canvas you need.

Note: This documentation is for version 1.3. Features may differ in older versions.

Core Concepts

To use MinePaint effectively, you should understand the following concepts:

  • Canvas: Your drawing area. You can resize it up to 300x300.
  • Layers: You can stack blocks on top of each other using the Layer System.
  • Blockstates: Many blocks (like Logs or Stairs) have different rotations.
