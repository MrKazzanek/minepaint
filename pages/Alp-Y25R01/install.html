How to Download

To download the program, go to https://minepaint.netlify.app/#download, select the appropriate version, and run the downloaded MinePaint.exe file.

Warning! Version Alp-Y25R01 does not include an installer, while newer versions do. To learn how the installation process works in newer versions, please refer to the documentation for those versions.

  • Memory: at least 500 MB
